# TEST UNIVESP

## Comandos principais do Git
  - Copiar repositório:
  ``` git clone https://github.com/matthewsbrandan/test-univesp.git ```
  - Iniciar git
  ``` git init ```
  - Adicionar alterações
  ``` git add . ```
  - Salvar alterações criando um marco no tempo
  ``` git commit -m "Mensagem explicando o nome da marcação" ```
  - Subir alterações
  ``` git push ```
  - Pegar alterações
  ``` git pull ```
